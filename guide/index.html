# 使用指南

随着CopyTranslator的更新，功能也不断丰富，不同版本之间的差异也逐渐显著，因此不同版本的使用指南是分开的，请根据您使用的版本来选择使用指南。

  • CopyTranslator v11 扶摇：这是一次创造性的大更新，用户使用体验可以说是飞跃，只能说非常爽，建议用这个(windows与mac)
  • CopyTranslator v10 破晓: 较为稳定，仅有windows版本
  • CopyTranslator v9 寒渐：支持windows,mac,linux，但不完全支持，已经年久失修
  • CopyTranslator v8.4.0
  • CopyTranslator v8.3.0
  • CopyTranslator v7 (opens new window)：仍支持32位系统，仅支持windows

更多的使用技巧，例如丝滑无比的galgame机翻技巧可以在使用技巧找到
