# Linux

提示

无法下载？尝试 国内如何下载 GitHub release 中的内容？ - Leoy的回答 - 知乎

注意事项

  • 推荐使用AppImage安装，可获得后续自动更新。

  • 如果无法启动，尝试在启动的时候添加参数--no-sandbox，如：copytranslator --no-sandbox，同时请保证libpng版本不低于1.6

  • 如果出现问题，请在终端下执行 /opt/copytranslator/copytranslator，查看问题所在

  • 启动应用时出现如图错误

    属正常现象，在终端执行 sudo usermod -a -G input $USER 即可解决
