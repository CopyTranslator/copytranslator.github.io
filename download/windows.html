# 通过包管理器安装

这些源的更新一般会稍微延迟几个小时，因为不是我直接更新的

  1. winget (opens new window)：执行命令 winget install copytranslator
  2. scoop (opens new window)：添加extra bucket(scoop bucket add extras)然后scoop install copytranslator即可，更新软件的话使用scoop update copytranslator
