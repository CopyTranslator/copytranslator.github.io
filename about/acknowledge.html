# 开源库

所基于的开源库太多，难以罗列，在这里仅列出主要的几项，同时也感谢其他默默无闻为开源事业做出贡献的人们：

  • Electron：使用 JavaScript, HTML 和 CSS 构建跨平台的桌面应用
  • Vue：渐进式 JavaScript 框架
  • Element: 一套为开发者、设计师和产品经理准备的基于 Vue 2.0 的桌面端组件库
  • OpenTranslate: 开放翻译标准库-核心翻译引擎
  • iohook: 捕获全局鼠标和键盘事件
  • robotjs: Node.js Desktop Automation.
  • eazydict 简单易用的命令行词典-核心查词引擎
  • node-mouse: Linux下捕获全局鼠标事件

# 友情软件

CopyTranslator从这些软件中学到了很多，以下均为非常优秀的软件，推荐大家尝试。

  • 沙拉查词：沙拉查词-聚合词典划词翻译
  • 划词翻译：让浏览任意语言的网站变得无比轻松的浏览器扩展程序。
  • 达达查词：比较好看的 Chrome 划词翻译(搜狗)插件，自带生词簿及吐司单词记忆，可与有道、扇贝单词同步
  • iciba-translate-userscript：iciba划词翻译 userscript
