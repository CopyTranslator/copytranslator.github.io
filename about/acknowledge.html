# 开源库

所基于的开源库太多，难以罗列，在这里仅列出主要的几项：

  • Electron：使用 JavaScript, HTML 和 CSS 构建跨平台的桌面应用
  • Vue：渐进式 JavaScript 框架
  • Element: 一套为开发者、设计师和产品经理准备的基于 Vue 2.0 的桌面端组件库
  • translation.js: 核心翻译引擎
  • iohook: 捕获全局鼠标和键盘事件
  • robotjs: 模拟键盘输入
