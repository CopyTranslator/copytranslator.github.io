# v10.0.0

欢迎反馈关于翻译格式，翻译效果不一致的问题。

# 更新内容

增强：

  1. UI重设计
  2. 使用Vuetify替换Element UI
  3. 支持暗黑模式 #164 (opens new window)
  4. 支持根据系统语言自动切换界面语言 #195 (opens new window)
  5. 使用ctrl+=与ctrl+-进行字体大小的缩放 #193 (opens new window)
  6. 优化查词时对句末逗号或句号的处理 #225 (opens new window)
  7. 可选的自动检查更新
  8. 增加局部快捷键ESC,可以隐藏窗口，修复windows上自动显示时无法自动获取焦点的问题。 #181 (opens new window)
  9. 添加俄罗斯语言，感谢@dEN5-tech (opens new window)
  10. 全局单例检查：以前可以同时开多个，现在只能开一个 #219 (opens new window)
  11. 优化对双屏的支持：#188 (opens new window)
  12. 百度OCR支持检测语言和更多语言 #249 (opens new window)
  13. 增加开机启动选项 #263 (opens new window)
  14. 增加toast提示当前状态 #275 (opens new window)
  15. 命令模式: 可以直接在原文框中输入动作，并按Ctrl+P执行该动作。

修复：

  1. 修复#250 注册百度OCR API后使用OCR翻译图片时报错（mac）
  2. 修复windows桌面图标及托盘图标不清晰的问题。
  3. 修复关闭监听剪贴板后拖拽复制仍生效的问题。
  4. 修复直接关闭窗口后设置没有保存的问题。
  5. 修复右键托盘图标，点击其他地方，菜单不消失 #152 (opens new window)
  6. 修复Windows10中用ALT+TAB切换窗口后copytranslator的窗口位置会有微弱的偏移 #176 (opens new window)
  7. 修复繁体中文检测成简体中文的问题 #215 (opens new window),#245 (opens new window)
