# v0.0.8 Zouwu pre-release

# 新特性

  1. 使用Electron+TypeScript+Vue实现，有了安装包，体积大了不少？

  2. 支持自定义界面语言：用户可以自行创建界面语言文件，CopyTranslator会自动检测

  3. 通过动作系统统一了各种事件，包括界面布局，菜单，快捷键，设置等。动作，包括切换各种开关，以及各种事件等等。

  4. 支持自定义界面样式/风格： 可以通过CSS 更改CopyTranslator的部分外观，如结果框，设置框，状态栏的字体，颜色，背景，边框等等。

  5. 自定义全局快捷键：几乎所有动作，都可以绑定全局快捷键。

  6. 自定义右键菜单/面板：不同模式的右键菜单不同，可以自行定义。几乎所有动作都可以用作菜单项。可用的快捷键，请查看Electron Accelerator。

  7. 字体缩放：两个模式均可通过Ctrl+wheel进行字体缩放。

  8. 优化贴边隐藏：现在可以隐藏到两边，并可以设置双击隐藏的方向/最小化。

  9. 自动黏贴：前提：选中自动复制，自动复制之后模拟ctrl+v自动黏贴，替换选中文字。

  10. 自动格式化：注意：此功能与自动复制冲突，会自动反选自动复制，将存在错误空行的剪贴板内容替换为正常的剪贴板

  11. 翻译来源切换：可以在Google,Baidu,Youdao,Sogou中选择

  12. 因为在很多应用当中都无法使用，现在点按复制默认是关闭的，需要的话，请手动打开。

  13. 新的交互方式，可以在设置在通知中看到翻译结果。
